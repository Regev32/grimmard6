Help

The purpose of this website is to provide imputations and probabilities for genotypes (full or partial).
The website receives the patient's HLA typing (as a gl string or separate alleles) and the patient's race as input.
Using py-ard and grimm, the website return the pateint's possible genotypes (and the probability for each) and the patient's possible haplotypes (and the probability for each according to race).
